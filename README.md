### Инструкция по установке
1. Открыть SQL - первое задание; 
1. Развернуть базу,  используя команду 
```
docker-compose up -d
```
1. Запустить приложение с помощью команды 
```
java -jar ./artifacts/app-deadline.jar
```
1. Запустить тесты
